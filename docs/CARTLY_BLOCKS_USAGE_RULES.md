# Cartly Blocks Usage Rules

## Home Page
Allowed:
- hero (static, carousel, text, grid, image)
- image_text
- bullets
- image
- faq
- featured_products
- categories
- testimonials

Restrictions:
- Hero allowed ONLY on Home
- First Hero slide gets H1

---

## Standard Pages
Allowed:
- text
- image_text
- bullets
- image
- faq

Restrictions:
- NO hero blocks
- NO product/category blocks
- Content-focused only
